### This is...

... a simple program to calculate terms in the "nim-world".

Parenthesises aren't supported yet (v0.1).

The code is ugly, inefficient and undocumented, but it does the job.

### Run on...

#### ...Windows:

##### Required files:
- Nim.exe
- msvcp120d.dll
- msvcr120d.dll

##### Run:

`Nim.exe`


#### ...Linux (Debian, Ubuntu):

##### Required files:
- nim_calc

##### Run:

`./nim_calc`


#### ...Linux (Any):

##### Required files:
- nim_calc.cpp

##### Run:
```
make
./nim_calc
```


#### ...Android:

##### Required files:
- NimCalc.apk

##### Run:

- Enable installing non-playstore apps (Settings->Security->"Unknown Source")
- Install .apk app
